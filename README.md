### getting started

```bash
# clone template
git clone https://github.com/maxhu08/neutrino-template.git

# cd
cd neutrino-template

# installing dependecies
npm install

# delete git info so you can add your own repo
rm -rf .git
```

then start the dev server:

```bash
# start server
npm run dev
```
